# RUBIK Pi Linux System 开放网络端口

| 端口号 | 协议 | 监听地址 | 服务/进程 | 功能描述 |
|--------|------|----------|-----------|----------|
| 1883 | TCP | 0.0.0.0 & ::: | mosquitto | MQTT 消息代理服务器。用于物联网设备与服务器之间发布/订阅消息通信。 |
| 53 | TCP | 127.0.0.1 & ::1 | dnsmasq | DNS缓存和本地解析服务。通常为本地其他服务（如容器、本地网络发现）提供域名解析。 |
| 33869 | TCP | 127.0.0.1 | containerd | 容器运行时守护进程的API接口。用于管理（启动、停止）容器。 |
| 111 | TCP | 0.0.0.0 & ::: | init (rpcbind) | RPC 端口映射器。为NFS（网络文件系统）等远程过程调用服务提供端口管理。 |
| 80 | TCP | 0.0.0.0 | lighttpd | 轻量级Web服务器。用于提供设备的Web配置界面或API接口。 |
| 22 | TCP6 | ::: | init (sshd) | SSH 远程安全登录服务。用于远程维护、调试和文件传输。 |

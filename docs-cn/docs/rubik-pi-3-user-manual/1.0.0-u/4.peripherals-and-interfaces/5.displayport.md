# DisplayPort

魔方派 3 拥有 1 个 USB Type-C 接口的 DisplayPort，如下图 5。

DP 的参数如下：

* 3840 × 2160 分辨率 @ 60 fps

* 单流传输 (Single stream transport)

* DisplayPort 和 USB 3.0 的并发功能

![](../images/image-139.jpg)

:::note
>
> 在 Ubuntu24.04 V1.0.0 版本中，暂不支持 DP 功能，DP 功能将在后续的版本中支持。
:::
